Getting Started

Welcome to the VCS API! This guide will help you make your first API call. To begin, you'll need to obtain an API key from your dashboard. Once you have your key, you can start making requests to our endpoints.

Installation (Conceptual Python Client)

pip install vcs-client

Basic Usage

from vcs_client import VCSClient

# Initialize client with your API key
client = VCSClient(api_key="YOUR_API_KEY")

reference = "The sun sets over the mountains, painting the sky in hues of orange and purple."
generated = "As evening approached, the mountains were silhouetted against a fiery sky."

try:
    scores = client.calculate_vcs(reference_text=reference, generated_text=generated)
    print(f"VCS Score: {scores.vcs_score}")
    print(f"GAS: {scores.gas_score}, LAS: {scores.las_score}, NAS: {scores.nas_score}")
except Exception as e:
    print(f"Error: {e}")

Ensure your development environment is set up with necessary HTTP client libraries if you are not using a dedicated client library.

Endpoints

POST /v1/calculate

Calculates the full Video Comprehension Score (VCS) along with its sub-components (GAS, LAS, NAS) based on a reference text and a generated text.

  • Method: POST
  • URL: https://api.vcs-metric.com/v1/calculate
  • Content-Type: application/json
  • Authentication: Required (Bearer Token)

Request Body:

{
  "reference_text": "The quick brown fox jumps over the lazy dog. It then finds a comfortable spot to nap.",
  "generated_text": "A fast brown fox leaped above the tired canine. Afterwards, it located a cozy place for a snooze.",
  "config": { 
    "language": "en",
    "embedding_model": "default_transformer_v2",
    "local_chronology_tolerance": 0.1 
  }
}

Response (200 OK):

{
  "vcs_score": 0.873,
  "gas_score": 0.915,
  "las_score": 0.820,
  "nas_score": 0.891,
  "components": {
    "sas_score": 0.850,
    "nas_d_score": 0.900,
    "nas_l_score": 0.880
  },
  "processing_time_ms": 120,
  "request_id": "req_abc123xyz789"
}

Rate Limits

To ensure fair usage and stability of the service, API requests are rate-limited. The current limits are typically:

  • Standard Tier: 100 requests per minute per API key.
  • Standard Tier: 5,000 requests per day per API key.
  • Premium Tiers: Higher limits available (contact sales for more information).

Exceeding these limits will result in a 429 Too Many Requests error. Check the Retry-After header (in seconds) or X-RateLimit-Reset header (UTC timestamp in seconds) for when you can retry. Consistent abuse may lead to temporary key suspension.

Relevant response headers:

  • X-RateLimit-Limit: The rate limit ceiling for the current request window.
  • X-RateLimit-Remaining: The number of requests remaining in the current window.
  • X-RateLimit-Reset: The time at which the current rate limit window resets (UTC epoch seconds).

Authentication

All API requests must be authenticated using an API key. Pass your API key as a Bearer token in the Authorization header.

Authorization: Bearer YOUR_API_KEY

Example using curl:

curl -X POST https://api.vcs-metric.com/v1/calculate \\
     -H "Authorization: Bearer YOUR_API_KEY" \\
     -H "Content-Type: application/json" \\
     -d '{
           "reference_text": "Example reference.",
           "generated_text": "Example generated."
         }'

Keep your API key secure and do not expose it in client-side code or public repositories.

Error Codes

The API uses standard HTTP status codes to indicate the success or failure of a request. Here are some common error codes:

  • 400 Bad Request: The request was malformed, or contained invalid parameters. The response body will often include a more specific error message.
  • 401 Unauthorized: Missing or invalid API key. Ensure your Authorization header is correctly formatted and your key is active.
  • 403 Forbidden: You do not have permission to access this resource, or your API key does not have the required scopes.
  • 429 Too Many Requests: You have exceeded your API rate limit. Check the Retry-After or X-RateLimit-Reset headers.
  • 500 Internal Server Error: An unexpected error occurred on our servers. Please try again later. If the problem persists, contact support.
  • 503 Service Unavailable: The service is temporarily unavailable, usually due to maintenance or overload. Try again after a short delay.
